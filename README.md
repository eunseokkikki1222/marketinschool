# 🏫 MarketInSchool

교내 학생들을 위한 중고거래 앱 MVP입니다.  
로그인 없이 바로 상품을 등록하고 거래할 수 있습니다.

## 🌐 배포 주소

**http://marketinschool-app.azurewebsites.net**

## ✨ 기능

- 상품 목록 보기 (카테고리 필터 + 검색)
- 상품 등록하기 (제목, 가격, 설명, 연락처, 카테고리)
- 상품 상세 보기
- 상품 삭제

## 🛠 기술 스택

- **백엔드**: Node.js + Express
- **프론트엔드**: Vanilla HTML/CSS/JS (SPA)
- **배포**: Azure App Service (Free tier, Korea Central)

## 🚀 로컬 실행

```bash
npm install
npm start
# http://localhost:3000 접속
```
